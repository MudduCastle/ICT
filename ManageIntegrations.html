Data Sources and Integrations

The Data Sources and Integrations page allows you to view and manage all of your projects. Below is an overview to manage adding tools with data sources.


Configure Source Systems

Step 1: Once integrations are added, you will find all of your integrations and connected accounts by clicking Manage Integrations from the left navigation.


Adding an Integration

Click on “Add Source Systems” and add your data source details for Integrating your tool using authentication credentials.


Step 2: On the Integrations page you can do the following:

1. Click “Test Connection” to check the connectivity of your tool.

2. Click “Save” option to save the data sources details provided by you.

3. Click “Cancel” option to cancel the data sources details provided by you.

Generate API Token

To generate API Token for your tools, click on the links provided in “Data Source Systems” pop-up

Create an API token from your Atlassian account:

Log in to https://id.atlassian.com/manage/api-tokens

Click Create API token.

From the dialog that appears, enter a memorable and concise Label for your token and click Create.

Click Copy to clipboard, then paste the token to your script, or elsewhere to save

After Selecting the API Token provide the Login ID and the Api Token

Api Token can be generated using the help text provided above the test connection button

After filling all the above details click on the test connection *Next you can see your test connection status through a pop up on the top right corner (Test Connection successful/Test Connection Failed)

If Test Connection is Successful (Save button will be enabled) Click on the save button and the source system will be added to application

If Test Connection is Failed (Save button will not be enabled) Please check the above jira details and provide the correct details

Click on cancel button if you dont want to add the source system

Success

Green Tick mark appears while the connection is success


Request

Red exclamation mark appears while the data source systems connection requires more details

Edit an Integration

At any point you may edit/update an integration from the page.

Step 3: Click “Edit” option to update the data sources details saved by you.

Delete an Integration

At any point you may delete an integration from the Integration page. This will remove all data associated with that data source

Step 1. Select the integration you would like to delete and click “Remove / Delete” from the actions menu.

JIRA Integration

Jira Software is an agile project management tool that supports any agile methodology, be it scrum, kanban, or your own unique flavor. From agile boards to reports, you can plan, track, and manage all your agile software development projects from a single tool.

To add the JIRA source system follow the below steps:

1. Select source system type as JIRA in Dropdown

2. Under the key section enter a unique key

3. The tenant id will be updated automatically

4. In url provide the jira sourcesystem url (example: https://{domain-name}.atlassian.net/)

Under authentication we are supporting 4 Types:

1. No auth

2. API Token

3. Single sign on (In Progress)

4. OAuth (In Progress)

It helps automate the parts of software development related to building, testing, and deploying, facilitating continuous integration and continuous delivery.

It is a server-based system that runs in servlet containers such as Apache Tomcat
