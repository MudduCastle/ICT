Overview

The Dashboard provides an objective view of performance metrics of all your projects and serve as an effective foundation for further dialogue. Also your dashboard is a business intelligence tool used to display data visualizations in a way that is understood and accessible to everyone.

Dashboards use raw data from these sources, spreadsheets and databases to create tables, line charts, bar charts and gauges in a central dashboard that users can look at and immediately understand the metrics they are looking for. Data dashboards simplify end of month reporting by allowing users to communicate information at any time without hours of preparation and analysing.

Dashboards can automatically generate reports with its data, at anytime, anywhere. No longer do users need to gather, analyze and format data

SQATA Dashboard

SQATA is a specialized dashboard for Software Quality Assurance & Test Automation.

Features include:

Detailed metrics related to user stories, defects, test cases, test automation, and code quality

Jenkins interface to execute and monitor automation jobs.

Cucumber report integration.

Detailed graphs and charts showing metrics over time for specific projects or rolled up to portfolio or enterprise level

Extensive customization options allow users to configure source systems, map data to projects, and display just the widgets and metrics that they wish to see.

Drill-down data viewers which allow users to quickly zero in on specific areas of interest

Consolidated widgets and reports pull data from decentralized agile source systems illustrating trends and relationships which may not be evident when reviewing data from tools in isolation.

Prismatica allows managers to perform data analytics in order to identify problem areas and maximize team performance

Graphics and charts help visualize abstract concepts such as CI/CD pipeline, infrastructure components, and team dynamics

This dashboard also have a customization feature where you can customize your dashboard with desired widgets

We can also customize the data according to project wise portfolio wise along with the data range fliters


SQATA Dashboard supports a wide range of widgets

1. Donuts --> Jira/Rally/Sonarqube/Jenkins

2. Graphs --> Jira/Rally/BitBucket

3. Job status --> Jenkins

4. Commit Tracker --> BitBucket

5. Report --> Sonarqube

6. Features Count --> Jira
